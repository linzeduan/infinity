# AI 算力全链路瓶颈分析——先进封装基板 + EDA + HBM 制造链

> **核心问题**：AI 芯片的算力提升正在被哪些环节卡住？哪些最没有被市场充分定价？
>
> **核心判断**：市场 80% 的注意力在 CoWoS 封装和 GPU 本身，但**先进封装基板**和 **EDA/Chiplet 工具**是两个被严重低估的瓶颈。前者是物理产能瓶颈（交期比 CoWoS 还长），后者是设计瓶颈（极强定价权且无替代）。

## 二、聚焦①：先进封装基板（FCBGA / ABF / Glass Core）

### 核心逻辑

> **一句话**：所有人都在追 CoWoS，但 GPU 下面那片基板的交期比 CoWoS 还长，讨论的人少 80%。

### 物理逻辑

```
AI GPU (H100/B200) → 芯片尺寸 > 800mm² → 封装基板尺寸 ~80×80mm
  → 传统有机基板的物理极限：翘曲、线宽、层数、尺寸
    → 需要 ABF（Ajinomoto Build-up Film）或 Glass Core
    → ABF 产能紧缺（味之素垄断，产能扩张极慢）
    → Glass Core 是破局方向，但量产良率仍在爬坡
```

**关键判断**：每代 AI 芯片基板单价从 $50（H100）→ $80（B200）→ $120+（Glass Core），**升级不降价**。

### Q0-Q5 评分

| 维度 | 评分 | 判断 |
|------|------|------|
| Q0 瓶颈时间 | ✅ | AI GPU 封装基板交期 12-16 周，是供应链最长环节之一。Glass Core 2026-2027 切换窗口 |
| Q1 约束范围 | ✅ | 行业级——NVIDIA/AMD/Intel/Google TPU 都面临同样问题 |
| Q2 延续性 | ✅ | 芯片越大、层数越多 → 基板单价越高。一路向上 |
| Q3 TAM | $15-20B | AI 相关占比从 20%→40%。单 AI GPU 基板价值量 $50-120+ |
| Q4 壁垒 | ✅ | 认证 2-3 年 + 良率爬坡极慢 + MSAP/mSAP 工艺门槛高 |
| Q5 认知差 | ✅ 高 | 市场 80% 注意力在 CoWoS，不到 20% 在基板 |

### 定价权检验：7/12（中等偏强）

- 技术验证周期 3/3，Capex 壁垒 2/3，材料/IP 壁垒 1/3，客户锁定 1/3
- **有议价力但非极端**——产能紧张时强，但下游也有压价能力

### 代际穿越：5/5（极强穿越）

- 替代风险极低，行业级需求，每代升级附加值提升

### TAM 与价值量

| 指标 | 数值 |
|------|------|
| 全球 IC 载板 TAM (2025) | ~$15B → 2028E ~$30B (CAGR 20%+) |
| 单 AI GPU 基板价值量 | H100: ~$50 → B200: ~$80 → Glass Core: ~$120+ |
| Glass Core 渗透率 (2026E) | <5% → 2028E ~15-20% |
| 主要基板厂 ABF 毛利率 | ~25-35%，高端 40%+ |

### 公司映射

| 标的 | 代码 | 位置 | AI 敞口 | 核心逻辑 | 关注等级 |
|------|------|------|---------|----------|---------|
| **欣兴电子 / Unimicron** | 3037.TW | ABF #1 | ~30-40% | 最大 ABF 产能，NVIDIA/AMD 主力供应商，Glass Core 期权未被定价 | 🔴 重点 |
| **Ibiden** | 4062.T | ABF 高端 | ~25-35% | 日系技术最强，NVIDIA 高端线，PE 偏高但壁垒最高 | 🔴 重点 |
| **AT&S** | ATSV.VI | Glass Core | ~20-30% | 欧洲唯一高端基板厂，Glass Core 技术领先，弹性大风险也大 | 🟡 关注 |
| **Shinko / 新光电气** | 6967.T | #3 | ~15-20% | 日本精工旗下，弹性不如欣兴 | 🟡 关注 |
| **南电 / NanYa PCB** | 8046.TW | ABF 扩张 | ~15-20% | 台塑旗下，弹性一般 | ⚪ 观察 |

### 与 AMB 基板的区别

> [[SiC_MOSFET封装技术_卡脖子分析_提纯]] 中的 AMB 基板是用于**SiC 功率模块**的陶瓷基板（高压大功率场景），本文讲的是**AI 计算芯片的 FCBGA/ABF 有机基板**。同属"先进封装基板"大方向，但技术和客户完全不同。

## 三、聚焦②：EDA + Chiplet 设计工具

### 核心逻辑

> **一句话**：芯片从单片变成多 Chiplet，设计复杂度指数级上升。市场当 Synopsys/Cadence 是半导体周期股，但它们其实是"AI 芯片复杂度杠杆股"——Chiplet 化后 EDA 附加费 +30-50%，且无替代竞争。

### 物理逻辑

```
单芯片 AI ASIC → 光罩尺寸限制(~800mm²)
  → 必须拆成 Chiplet → 多芯片协同仿真需要新 EDA
  → AI-driven design automation 提升效率
  → 所有芯片公司都需要 EDA——这是"芯片设计的水和电"
```

### Q0-Q5 评分

| 维度 | 评分 | 判断 |
|------|------|------|
| Q0 瓶颈时间 | ✅ | Chiplet 已成标准架构，Synopsys 3DIC Compiler 是关键工具 |
| Q1 约束范围 | ✅ | 行业级——NVIDIA/AMD/Broadcom/Qualcomm 无一例外 |
| Q2 延续性 | ✅ | 芯片越复杂 → EDA 价值越高，每代授权费上涨 |
| Q3 TAM | $20B+ | Synopsys $7B+ / Cadence $5B+，80% 毛利率 + 经常性收入 |
| Q4 壁垒 | ✅ | 双寡头垄断 80%+ 份额，迁移成本极高 |
| Q5 认知差 | ✅ 高 | 市场归类为"半导体工具"，忽略"AI 复杂度杠杆" |

### 定价权检验：12/12（极强定价权）

- **满分**——切换周期 3-5 年 + 构建全流程需 $10B+ 研发 + 数千项算法专利 + 80% 经常性收入
- 全球仅两家（Synopsys + Cadence）真正全流程 EDA

### 代际穿越：5/5

- 30 年未被动摇，零替代风险，行业级需求，每代升级增值

### TAM 与价值量

| 指标 | 数值 |
|------|------|
| 全球 EDA 市场 (2025) | ~$18B → 2028E ~$25B (CAGR 12%+) |
| Synopsys 收入/毛利率 | ~$7B / 80%+ |
| Cadence 收入/毛利率 | ~$5B / 78%+ |
| 3DIC/Chiplet EDA 附加费 | 传统授权费 +30-50% |

### 公司映射

| 标的 | 代码 | 位置 | AI 敞口 | 核心逻辑 | 关注等级 |
|------|------|------|---------|----------|---------|
| **Synopsys** | SNPS | EDA #1 | ~15-20% | 3DIC Compiler + Synopsys.ai（AI-driven EDA）双重暴露，+20-40% 上行 | 🟡 关注 |
| **Cadence** | CDNS | EDA #2 | ~15-20% | Integrity 3DIC 平台 + 收购 Ansys 增强仿真能力 | 🟡 关注 |
| **ARM** | ARM | IP | ~10-15% | IP 授权受益 chiplet（AMBA CHI 协议），非 EDA 但生态协同 | ⚪ 观察 |

> 注意：EDA 不是"物理产能瓶颈"，而是**设计瓶颈**——没有它任何芯片都设计不出来。属于"软瓶颈"但拥有整条链上最强的定价权。

## 四、其他值得跟踪的方向

### HBM 制造链（已被部分定价）

HBM 本身是已知瓶颈，但其中 **Hybrid Bonding（混合键合）设备** 的共识差中等偏高：
- HBM4 将从微凸点切换到混合键合（HB）
- 关键设备：**Besi (BESI.AS)**、ASMPT (0522.HK)
- 建议跟踪但不深度分析：等待 HB 在 HBM4 中明确的量产时间表

### Chiplet 测试（KGD / 多芯片系统测试）

- TAM 小（$3-5B）但增速极快（CAGR 20%+）
- 关键公司：**Teradyne (TER)**、Advantest (6857.T)
- 共识差：高——很少人想到 chiplet 繁荣的最大受益方之一是测试设备
- 建议跟踪：等待更明确的 KGD 测试标准

## 六、关键结论

1. **当前最该关注的新增方向**：先进封装基板（FCBGA/ABF/Glass Core）——比 CoWoS 更少人讨论，交期更长，壁垒不低。**欣兴电子 3037.TW** 和 **Ibiden 4062.T** 是最确定的标的。

2. **定价权最强但被错配分类**：EDA（Synopsys / Cadence）——市场当它们是"半导体周期股"，但 chiplet + 3DIC + AI-driven EDA 三重杠杆应让它们享有 AI 估值。

3. **三条链打通后的全貌**：「电进去」（800V HVDC）→「算出来」（算力链）→「热带走」（液冷），三链互联，跨链交叉受益的公司最值得优先关注。
