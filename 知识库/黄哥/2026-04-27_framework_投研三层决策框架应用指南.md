
# 投研三层决策框架应用指南

来源：20260427-投研方法论-芒格与李录.pdf  
分析时间：2026-04-27 | 核心判断：三层框架本质上是从"快速筛选→深度验证→长期托付"的递进式风险控制，而非简单的分析步骤

---

## 一、框架核心架构

### 三层方法论对标表

| 层次 | 目的 | 核心问题数 | 决策形态 | 时间成本 | 风险容度 |
|--|--|--|--|--|--|
| **基础五问** | 快速筛选 | 5个 | 是/否 二选一 | 30分钟 | 高风险（误杀或误放）|
| **芒格三问** | 深度理解 | 3个 | 降级/PASS/深化 三态 | 3-7天 | 中等风险 |
| **李录五问** | 长期持有 | 5个 | 入池/观察/放弃 多态 | 2-4周 | 低风险（托付决策）|

**核心逻辑**：每一层都是对前一层的"质疑"，而非"补充"
- 基础五问答YES，不意味着能买入
- 芒格三问通过，不意味着适合长期
- 李录五问入池，才能考虑动真金白银

---

## 二、三层框架的机制解析

### 第一层：基础五问（快速筛选）

**实战地位**：垃圾箱过滤器

**五个问题的递进逻辑**：

```
问题1：卖什么能力？
  └─ 目标：排除"弄不懂业务"的公司
  
问题2：为什么赚钱？（壁垒）
  └─ 目标：排除"无护城河"的公司
  
问题3：利润在哪？（结构）
  └─ 目标：排除"利润虚幻"的公司（如麦当劳的地产+加盟模式）
  
问题4：增长驱动？
  └─ 目标：区分"行业红利"vs"自身竞争力"
  
问题5：持续多久？（周期）
  └─ 目标：初步判断投资周期（交易vs投资）
```

**通过标准**：五问都有明确答案 → 进入第二层

**失败信号**：
- 任何一问无法用一句话说清 → PASS
- 利润来源模糊不清 → PASS
- 增长只是行业红利，公司贡献不到20% → 降级观察

**实战陷阱**：
❌ 错误做法：用基础五问的答案作为最终投资决策
✅ 正确做法：基础五问只是"值不值得花2周深研"的门槛

---

### 第二层：芒格三问（深度理解）

**实战地位**：真伪识别器

**核心哲学**："Invert, always invert" — 先想死法，再想活法

#### 第一问：我真的懂这个生意吗？

**操作标准**：能否用**最简单的话**把生意讲给一个普通人听清楚

```
判断标准：
✅ 光模块 = "让数据传得更远更快"
✅ DC-DC = "把电变成设备能用的样子"
❌ 复杂的衍生品定价模型 = 无法简化 → 不投
```

**关键洞察**：
- 复杂不等于深刻
- 无法用简单话解释 = 要么是自己没懂，要么是这个生意本身就有问题
- "听起来懂了" ≠ "真的懂了"，判断标准是能否预测管理层的下一步行动

**实战应用**：
写出三个版本的解释：
1. 给中学生讲（最简）
2. 给行业外的投资者讲（中等）
3. 给行业专家讲（完整）

如果任何一个版本讲不清，说明理解还有漏洞。

---

#### 第二问：这个生意10年后还在吗？

**操作标准**：逆向工程法 — 先写出3种"死法"

```
三种死法模板：
1. 技术替代 → CPO会不会替代光模块？
2. 架构替代 → 系统变化会不会绕过它？
3. 整合淘汰 → 会不会被更大的系统吞掉？
```

**评估规则**：
- 1种死法成立 → 该标的从"长期持有"降级为"交易"
- 2种死法成立 → PASS，不看了
- 0种死法成立 → 继续深化，进入第三问

**实战案例**：
| 标的 | 死法1 | 死法2 | 死法3 | 结论 |
|--|--|--|--|--|
| NVIDIA | 无 | 无 | 无（AI无替代品） | 核心资产 |
| 某小光模块厂 | 有（CPO） | 有（架构） | 无 | 中期风险 |
| 某消费热点 | 无 | 有（风口过） | 有（替代品） | PASS |

**关键对话**：如果这个公司消失了，世界会出问题吗？
- 会 → 这是核心资产
- 不会 → 可替代，溢价空间有限

---

#### 第三问：这个公司有没有"不可动摇的位置"？

**操作标准**：系统依赖性测试

```
四个维度：
1. 客户离不离得开你？ → 替代成本多高？
2. 替代成本有多高？ → 切换风险多大？
3. 你是不是系统的一部分？ → 被集成的风险？
4. 竞争对手想干掉你难不难？
```

**经典案例**：
- **Microsoft** = "离不开" → Office+Windows生态锁定
- **Apple** = "舍不得" → 生态+隐私承诺+用户粘性
- **某个创新型小公司** = "可以换" → 没有结构性优势

**实战应用**：
问一句狠话：如果我是竞争对手，我会怎么干掉这个公司？
- 如果想不到办法 → 位置不可动摇
- 如果办法很多 → 警惕，仓位要小

---

**芒格三问通过标准**：
1. 一句话说清生意 ✅
2. 找不到两种及以上的死法 ✅
3. 有明确的系统性位置 ✅

→ 进入第三层（李录五问）

---

### 第三层：李录五问（长期持有）

**实战地位**：托付认证系统

**核心特性**：不是"看这个能涨多少"，而是"看这个能不能拿5年"

#### 第一问：这是不是一个"值得长期理解的世界"？

**判断标准**：区分"时代主线"vs"时代噪音"

```
时代主线（可长期持有）：
✅ AI算力 / 电力 / 带宽 → 信息爆炸的必然需求
✅ 芯片制造 / 数据中心 → 底层基础设施
✅ 医疗健康 → 人口老龄化长期需求

时代噪音（短期交易）：
❌ 某个消费热点 → 情绪周期，1-2年消退
❌ 风口产业 → 政策驱动，可能反转
❌ 题材板块 → 缺乏长期需求逻辑
```

**识别方法**：
问题：这个需求是被"创造出来的"还是被"需要的"？
- 被需要 → 长期刚需，可以托付
- 被创造 → 政策/情绪驱动，有风险

---

#### 第二问：这个公司是在"顺势"还是在"借势"？

**核心区分**：

```
顺势公司：行业发展，它自然变强
  例：NVIDIA在AI时代
  特征：管理层不需要做什么，就自然增长
  
借势公司：行业好，但它只是蹭风口
  例：某个小光模块厂在AI热潮中
  特征：需要不断创新和市场拼杀才能维持
```

**李录的金律**："行业变大，它必然变强"

**实战应用**：
问管理层假设：如果行业增速从50%下降到10%，公司会不会陷入困境？
- 会 → 借势公司，风险高
- 不会 → 顺势公司，可以长期托付

---

#### 第三问：这家公司赚的是"真钱"还是"结构性的钱"？

**核心区分**：

```
真钱（可持续）：
✅ 平台型公司的利润 → 来自结构优势，生命周期长
✅ 科技壁垒的利润 → 来自技术积累，难以复制

结构性的钱（周期性）：
❌ 猪周期的利润 → 来自价格周期，高点还要出
❌ 资源类利润 → 来自大宗商品周期，有顶
```

**判断标准**：
剔除一个好年份，公司的正常利润有多少？
- 如果剔除好年份，利润腰斩 → 结构性收益，谨慎
- 如果剔除好年份，利润基本不变 → 真实收益，可以持有

---

#### 第四问：如果市场关闭5年，我还敢不敢持有？

**本质问题**：把股票当成"企业所有权"而非"交易筹码"

**三个子问题**：
1. 现金流是否真实？
2. 生意是否稳定？
3. 管理层是否可靠？

**实战逻辑**：
这个问题强制你停止关注股价、停止看技术面，只看基本面。
- 如果你的答案是"不敢"→ 这不是长期持有品种
- 如果你的答案是"敢，很敢" → 可以进入持有池

**关键信号**：
```
危险信号：
❌ "虽然基本面一般，但短期有催化剂"
❌ "虽然管理层不靠谱，但技术面强"
❌ "我看不太懂，但听说会涨"

安全信号：
✅ 即使5年不涨，我也愿意每年收这个现金流
✅ 即使股价腰斩，我还会加仓
✅ 我能清楚说出，为什么管理层的下一步行动是什么
```

---

#### 第五问：我是不是在"能力圈之外"？

**判断标准**：四层递进

```
第一层：我是不是只是"听起来懂了"？
  → 复述别人的话，vs 有自己的理解差异

第二层：我有没有一手信息理解能力？
  → 能否获取和理解行业第一手数据

第三层：这个东西复杂到需要"信别人"吗？
  → 如果需要完全相信某个分析师，基本不重仓

第四层：我的优势在哪里？
  → 相比其他投资者，我为什么更了解这个领域？
```

**实战规则**：
如果超过第二层都答不了，这个标的就不适合你。

---

**李录五问通过标准**：
1. 这是时代主线，不是噪音 ✅
2. 公司在顺势而非借势 ✅
3. 利润是真实的结构性收益 ✅
4. 我敢在无流动性情况下5年持有 ✅
5. 这在我的能力圈内 ✅

→ **确认：可以入持有池，等待合理价格**

## 三、三层框架的常见陷阱

### ❌ 陷阱1：跳过第一层直接深研

**症状**：看到一个热点，直接用芒格三问深研

**问题**：浪费时间研究本来就不该进的标的

**规避方法**：
- 基础五问必须是自动化的快速筛选（30分钟完成）
- 即使很有把握，也要严格执行五问，避免遗漏

---

### ❌ 陷阱2：基础五问答YES就下单

**症状**：表面上五问都过了，就觉得可以买

**问题**：基础五问覆盖90%基本面，但遗漏了10%的致命风险

**规避方法**：
- 明确告诉自己：基础五问通过 = "这个东西值得花2周研究"，不是"可以买"
- 进入第二层前，必须有等待期

---

### ❌ 陷阱3：被芒格三问的简洁性迷惑

**症状**：三个问题听起来很简单，就简化了判断过程

**问题**：简洁不等于容易，每个问题都需要2-3天的深思

**规避方法**：
- 每个问题都要产出一份"判断记录"
- 写下推理过程，而不是只记结论
- 3个月后回看记录，看自己当时的推理有没有漏洞

---

### ❌ 陷阱4：对李录五问的误读

**症状**：把"能拿5年"理解为"必然5年不动"

**问题**：李录五问的意思是"我有底气持有5年"，不是"我一定要拿5年"

**规避方法**：
- 买入后仍需每季度复盘：条件有无变化？
- 一旦逆序出现第二问的"死法"，需要立即重估
- 持有不是惯性，而是持续的信心确认

---

### ❌ 陷阱5：混淆"基本面好"和"股价便宜"

**症状**：用三层框架确认基本面后，就认为价格必然合理

**问题**：框架验证的是企业质量，不是估值合理性

**规避方法**：
- 李录五问通过后，结论是"进入跟踪清单"，不是"买入"
- 需要再加一步：等待"合理价格"的出现
- 好公司也要等好价格，这是最后一道防线

## 四、框架的失效场景

| 失效场景 | 具体例子 | 规避方法 |
|--|--|--|
| 黑天鹅事件 | 突然政策反转 / 供应链中断 | 定期压力测试，最坏情景演推 |
| 管理层作恶 | 财务舞弊 / 挪用资金 | 关注内部人增减持，定期审计 |
| 赛道消灭 | 整个行业被政策封杀 | 跨越多个周期观察，不赶风口 |
| 竞争格局反转 | 新对手凭借新技术颠覆 | 保持对行业动态的关注，不要闭眼持有 |

## 五、与黄哥决策体系的关联

### 框架对标：

| 黄哥系统 | 三层框架对应 | 核心差异 |
|--|--|--|
| "能力在哪" | 基础五问Q2（壁垒） | 黄哥问的是整体竞争力，框架问的是单一壁垒 |
| "钱从哪来" | 基础五问Q3（利润结构） | 完全一致 |
| "周期识别" | 基础五问Q5 + 李录Q1 | 框架更关注周期的长期趋势，黄哥更关注当前位置 |
| "系统性位置" | 芒格Q3（不可动摇） | 完全一致，都强调结构性优势 |

**融合使用逻辑**：
```
黄哥关注："现在这个标的在周期的哪个位置？"
框架答：用三层方法论判断这个标的是否值得关注

黄哥的优势：短周期操作 + 风险管理
框架的优势：中长期持有 + 企业质量评估

结合用法：用框架选品，用黄哥逻辑选时
```

## 六、实战操作checklist

### 当你看到一个新标的时：

```
【第0步】停止继续看，完成以下清单再讲话

【基础五问】(30分钟内)
□ 这个公司卖的是什么能力？（一句话）
□ 为什么能赚钱？（列出1-2个壁垒）
□ 利润真的来自这些地方吗？（举例验证）
□ 增长靠什么驱动？（行业/份额/价格分解）
□ 这个生意能持续多久？（估计周期）

判断：五个都答清楚 → 进第二步，否则PASS

【芒格三问】(3-7天)
□ 一句话把生意讲给普通人听，讲了几遍才讲清？
□ 写出3种死法，哪些成立？
□ 这个公司在竞争中的位置是什么？别人想干掉它难不难？

判断：能简洁讲清 + 无法找到致命死法 + 位置稳固 → 进第三步

【李录五问】(2-4周)
□ 这个需求是长期刚需还是短期情绪？
□ 公司在顺势还是借势？
□ 利润来自周期还是结构性优势？
□ 5年无流动性，我还敢持有吗？
□ 这真的在我的能力圈内吗？

判断：全部YES → 入池，等待好价格；有NO → 重新评估或PASS

【定期复盘】(每季度一次)
□ 当初的判断依然成立吗？
□ 有无新的风险信号出现？
□ 需要调整仓位或卖出吗？
```

## 七、最后的对抗视角

**这个框架最可能的失效点**：

❌ **假设1**：框架假设你有足够的时间做深研（实际很多人没有）

→ 规避：对于时间紧张的投资者，只用基础五问快速排序，重仓配置给懂得少的头部公司

❌ **假设2**：框架假设"复杂 = 不懂"（但某些复杂的东西确实值得研究）

→ 规避：区分"无法解释的复杂"（如衍生品定价）vs"需要时间学习的复杂"（如芯片工艺），前者PASS，后者可以投时间

❌ **假设3**：框架过度依赖"逆向思维"（有时正向思维也很重要）

→ 规避：用三种思维方式看一个标的——顺向（为什么会成功）、逆向（为什么会失败）、横向（对比竞品），三者都成立才下决定
