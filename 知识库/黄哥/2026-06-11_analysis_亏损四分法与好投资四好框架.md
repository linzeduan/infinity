# 亏损四分法与好投资四好框架

来源：260610亏损不是结果问题，而是对下注理由的回测。.pdf  
分析时间：2026-06-11 | 核心判断：黄哥将"止损决策"从情绪事件转化为可复盘的理性验证流程；技术面由"嗤之以鼻"到"逐帧学习"是本篇最重要的模型演进信号

---

## 核心判断（一行）

亏损分类框架的本质是把"我应不应该割肉"这道情绪题，重新变成"我的下注理由是否还成立"这道验证题——复盘能力比止损纪律更根本。

---

## 三栏增量分析

### 一、新增原则

**亏损四分法**（对已有模型"风控"模块的结构性补充）：

| 亏损类型 | 判断标准 | 处置原则 |
|----------|----------|----------|
| 基本面亏损 | 订单逻辑证伪/认证失败/量价齐跌 | 立即离场，买入理由已消失 |
| 估值亏损 | 公司无误，但买贵了（AI溢价下20倍PE→60-80倍） | 检查"我买的是公司本身还是市场继续给溢价的期权"；溢价收缩时杀估值无需基本面变差 |
| 情绪/技术面亏损 | 基本面和估值均合理，资金轮动导致下跌 | 可接受、可解——格雷厄姆"市场先生低价报价"框架下的买入机会 |
| 仓位亏损 | 同一标的，5%仓位跌20%可理性复盘；40%仓位跌20%触发情绪失控 | 仓位本质是心理承受力问题，不是数学问题 |

**好投资四好框架**（对"铲子投资五层框架"的底层补充）：

```
好公司 → 值不值得研究（能不能买）
好价格 → 底座价值 + 成长价值 vs. 趋势期权溢价
好时机 → 基本面/资金/板块三重验证入场节点
好仓位 → 不影响情绪 + 留有补仓空间 + 错了不影响组合
```

**止损三分法**（与亏损四分法对应）：
- 基本面止损 → 理由证伪，果断离场
- 估值止损 → AI溢价收缩，市场从"讲未来"转向"看当期利润"时高PE危险
- 技术/仓位止损 → 跌破技术位、板块趋势破坏、或仓位让你"为了证明自己对而非赚钱而持有"

### 二、强化或修正的已有原则

**强化：安全边际**（已在模型中，此篇再次强调）  
追高时必须问"我的安全边际在哪里？"——芒格原话被直接引用："一个没有兑现的期权，其价值就是零。"这是黄哥用来对抗AI溢价追高冲动的核心防御。

**强化：估值期权意识**  
20倍PE传统业务 + AI故事 → 60-80倍甚至更高，买入时必须明确自己买的是哪一层。与已有模型中"AI大地主困局"文章的逻辑高度一致：溢价层一旦叙事退潮，跌幅来自期权溢价而非基本面。

**重大演进：技术面从"辅助"升级为"与基本面并列"**  
黄哥原有模型：以基本面和宏观框架为核心，技术面仅作为辅助参考。  
此篇明确表态："我以前对K线嗤之以鼻，现在我对K线逐帧学习。"  
新立场：**技术面就是告诉你这场游戏资金站在哪一边**，可与基本面共同构建投资原则。  
→ 入场节点判断（好时机）：基本面有无恶化/好转 + 资金有无重新进来 + 板块是否止跌，三项并列。

---

### 三、与模型矛盾点

**容错率表述的内在张力**：  
黄哥一贯强调"高确定性才下注"（安全边际、能力圈原则）。但本篇用费德勒54%成功率类比投资，暗示接受相对高的失败率是正常的——"亏损构成了你的投资完整性"。  
两者并不矛盾，但侧重点不同：之前模型更多讲"入场前提高胜率"，此篇更多讲"入场后如何理性处理亏损"。这是同一系统的两个阶段，但如果混用，容易被市场参与者曲解为"54%就够了，不用追求高胜率"。  
→ 更精确的理解应是：提高入场胜率（安全边际）是前置条件，接受亏损是运营纪律，两者不可替换。

**反方视角**：亏损四分法的最大风险是"错误分类"——当一笔亏损实际上是基本面崩坏（应立即离场），持有者却将其归类为"情绪/技术面亏损"（可接受、等反弹），这种分类偏差会把止损纪律彻底架空，四分法反而成了拒绝割肉的借口。分类本身同样需要主观判断，而人在亏损中的判断系统性偏向于"我没错"。
