## STEP 2 · 决策原则提炼

### 原则①：从认知差入手，不买市场已知道的

- **触发条件**：当某环节（如CoWoS封装）已是市场共识热点，讨论量远超实际卡口程度
- **行动方向**：向上游递归，找同等重要但被讨论80%少的环节
- **该原则的反例边界**：若认知差来源于信息不对称而非结构性壁垒（例如某国政策短暂封锁消息），认知差修复后优势即消失，不能简单等同于长期机会

### 原则②：物理约束驱动的瓶颈优于政策/周期驱动的

- **触发条件**：分析链条中存在"物理极限"节点（材料特性、工艺良率、热管理上限）
- **行动方向**：此类瓶颈具代际穿越性，可以在周期低点耐心持有；用"代际穿越5/5"替代P/E估值
- **该原则的反例边界**：技术路径切换时（如玻璃基板颠覆ABF有机基板），物理约束转移，原来壁垒消失——需持续追踪Glass Core良率爬坡节奏

### 原则③：定价权检验先于TAM判断

- **触发条件**：识别到潜在投资机会
- **行动方向**：用12分制定价权框架评估（技术验证、Capex壁垒、材料/IP壁垒、客户锁定）。EDA满分12/12 = 核心仓；ABF 7/12 = 弹性仓；低于6分 = 不碰
- **该原则的反例边界**：产能极度紧张期，即使7/12的标的也可能爆发式上涨（供需比定价权更重要），错过短期机会是这条原则的成本

### 原则④：三条链打通看全局，优先跨链受益

- **触发条件**：分析AI基础设施整体投资布局
- **行动方向**：算力（封装/EDA）→电力（800V HVDC/SiC）→散热（液冷）三链均有覆盖，优先选跨链交叉受益标的（如Delta 2308.TW）
- **该原则的反例边界**：三条链节奏不同步时（算力Q3爆发/电力Q4跟进/散热Q1+2才落地），跨链逻辑可能导致持仓节奏错位

### 原则⑤：Q0-Q5框架系统筛选，防止主观叙事驱动决策

- **触发条件**：面对任何新的"AI投资机会"叙事
- **行动方向**：逐项打分（时间是否已到Q0、约束范围、延续性、TAM、壁垒、认知差），六维均✅才列为重点，否则降级为跟踪
- **该原则的反例边界**：早期阶段机会（如存算一体）刻意在Q2/Q3打低分，可能错过颠覆性机会；需设置"太早期跟踪池"单独维护

## STEP 3 · 因果链建模（升级版）

### 路径一：牛市/算力扩张期——基板认知差窗口

```
[NVIDIA新GPU发布，单片尺寸>800mm²] 
  → [权重：极高] 
  → [CoWoS封装产能成为显性瓶颈，市场开始追封装链] 
  → [叠加判断：基板交期12-16周比CoWoS还长，但媒体讨论度不到1/5] 
  → [行动：布局欣兴3037.TW + Ibiden 4062.T，等待市场从封装追到基板]
```

### 路径二：熊市/EDA被错误定价为周期股时

```
[半导体周期下行，市场将Synopsys/Cadence当周期股抛售] 
  → [权重：高] 
  → [叠加判断：Chiplet化是否减速？→ 不减速；AI芯片设计复杂度是否下降？→ 上升] 
  → [等待：确认EDA收入稳定性（80%经常性收入，非项目型）] 
  → [行动：逢低建仓，用AI复杂度杠杆逻辑重新定价而非P/E]
```

### 路径三：不确定期/Glass Core技术切换窗口

```
[Glass Core量产良率爬坡加速，>15%市场渗透率] 
  → [权重：中，需持续观察] 
  → [叠加判断：欣兴/Ibiden现有ABF路线是否被动？AT&S技术是否领先？] 
  → [等待：良率>50%且客户拿到认证才是切换信号] 
  → [对冲：欣兴仓位适当减少，增加AT&S观察仓]
```

## STEP 6 · 模型可靠性评估

**优势场景**（此思维在什么环境下最有效）：
- AI基础设施高速扩张期，物理产能瓶颈快速形成
- 市场注意力高度集中于某一显性环节，存在结构性认知差
- 可以用3年以上视角持有，不受季报扰动

**失效场景**（此思维在什么环境下会系统性失误）：
- 技术路径发生根本性跳变时（如量子计算颠覆传统芯片架构，物理瓶颈全部重写）
- 地缘政治切断供应链时（台积电、欣兴均在台湾，一旦台海局势激化，所有"物理壁垒"的分析都要重写）
- EDA公司被政府强制开源或分拆时（虽概率极低，但定价权12/12的前提是双寡头合法性）

**与其他流派的核心差异**：
- vs 趋势交易派：黄哥不追短期价格动量，追逐"物理约束时间窗口"
- vs 纯宏观派：他从产品级物理逻辑出发，不从利率/货币驱动推导
- vs 成长股派：他先问"定价权"，而非先问"增速"
