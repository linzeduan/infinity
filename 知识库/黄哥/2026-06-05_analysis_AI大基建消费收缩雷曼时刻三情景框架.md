## STEP 1 · 信息压缩

**高频核心变量**（情绪权重）：

| 变量 | 出现频率 | 情绪权重 |
|---|---|---|
| 能源成本 | 极高 | 负向（持续上行） |
| 逆全球化/关税 | 极高 | 负向 |
| AI资本开支（CapEx） | 高 | 中性偏负（耗资巨大但尚未兑现） |
| 消费收缩/L型分化 | 高 | 负向 |
| 雷曼时刻 | 中 | 负向（警示性类比） |
| 死亡螺旋 | 中 | 负向（AI降本→收入预期↓→消费↓） |

**核心结构认知**：黄哥用一句话概括了整个宏观框架——**"世界将进入一种高成本世界里的效率军备竞赛"**。这不是短期波动，而是结构性转变。

## STEP 2 · 决策原则提炼

**原则1：能源是一切成本的底层锚点**
- 触发条件：中东地缘风险持续、欧洲能源依赖未解除
- 行动方向：任何依赖廉价能源的商业模式需要重估
- 反例边界：战事快速缓和、中东和谈达成 → 能源回落可暂时解除压力

**原则2：逆全球化叠加 → 利润空间系统性收缩**
- 触发条件：美国301关税（60个经济体，最高12.5%）落地 + 欧盟扩大对华限制
- 行动方向：中国新三样（电车/光伏/锂电池）产业外移加速，迁移成本压缩利润
- 反例边界：贸易谈判实质性进展，关税部分豁免

**原则3：AI是重资产重工业，不是轻资产互联网**
- 触发条件：Alphabet单季CapEx $357亿、Meta全年$1250-1450亿
- 行动方向：AI上游估值逻辑需检验"下游现金流加速度何时超过资本开支流速"
- 反例边界：AI应用端大规模付费能力快速兑现 → 改变估值逻辑

**原则4：三重挤压形成下行螺旋**
- 触发条件：消费者被挤压 → 企业利润被挤压 → AI投资被挤压，链条依次传导
- 行动方向：此时需减少高估值、高beta仓位；寻找成本端受益标的
- 反例边界：政府财政强刺激直接托底消费端，打断螺旋传导

**原则5：苗头向坏情景发展时，止盈缩仓优先**
- 触发条件：黄哥明确表态"苗头正在向坏情景发生"
- 行动方向：适当止盈和缩减仓位
- 反例边界：好情景概率虽最低，但若数据反转（通胀回落+战事缓和），需快速切换立场

## STEP 3 · 因果链建模

**路径A（基准情景·当前状态）：类滞胀分化**
```
能源持续偏贵[高] + 欧洲通胀3.2%[高] → 权重评估：高
叠加：逆全球化关税落地[中] + 消费L型分化[高]
→ 判断：能源不便宜 + 通胀不低 + 利率下不来 + 消费不强
→ 行动：持仓继续谨慎，等待AI应用端兑现信号
```

**路径B（坏情景·概率上升）：三重断裂**
```
中东局势升级[触发] → 油价重新冲高[高] 
叠加：美国301关税大范围落地[高] + 欧洲扩大对华限制[高]
→ 判断：成本端↑ + 需求端↓ → AI上游从"高估值"进入"估值杀"
→ 行动：止盈缩仓，降低AI上游仓位
```

**路径C（好情景·概率最低）：全球化重启**
```
战事缓和[低概率] → 能源回落[低] + 301关税豁免[低]
叠加：AI应用端大规模付费兑现[低]
→ 判断：成本端↓ + 消费端↑ + 资本开支合理化
→ 行动：全面进攻，重配AI全链条
```

## STEP 4 · 认知偏好画像

**风险偏好**：情境切换型。
- 基准情景下：保守持仓，观察信号
- 信号明确偏向坏情景时：主动止盈缩仓（本文明确提示）
- 切换条件：是否出现"能源+关税+消费"三重叠加

**时间维度**：中长期结构判断为主（全球高成本结构性转变），短期信号作为"情景触发器"。两者权重约3:7（长期结构主导，短期信号触发操作）。

**决策锚点优先级**：
1. 宏观结构（逆全球化、能源、AI资本开支方向）
2. 叙事驱动（雷曼时刻类比、铁路修建类比）
3. 数据驱动（欧元区CPI 3.2%、Brent $94、CapEx数据）

**认知盲区**：本文对AI应用端兑现的时间节点几乎不预测——"下游的现金流加速度什么时候超过资本开支的流速"这个问题被提出但未解答。这是黄哥框架的系统性留白：他擅长识别结构性风险，但对技术应用具体落地时点保守估计。

## STEP 5 · 表达风格建模

**核心结构**：问题先行（"这件事最奇怪的地方是……"） → 逐层拆解（能源→逆全球化→AI重工业→三重挤压→中国困境） → 三情景收尾

**高频类比**：
- AI = 修铁路（重资产、大兴基建）
- AI = "中国制造"（通过规模摊平成本，赚差价）
- 从牛肉切到鸡肉（消费降级的具体化描述）
- 基板翘曲 = 乐事薯片（在下一篇封装文章出现的类比风格）

**绝对化表达**：偶尔使用（"苗头正在向坏情景发生"），但通常用三情景框架保留弹性，不做绝对预测。

**情绪基调**：警示型 + 冷静分析型并存。警示信号明确，但处置建议克制（"适当止盈"而非"全部清仓"）。

## STEP 6 · 模型可靠性评估

**优势场景**：
- 宏观逆风环境识别（滞胀、关税、能源冲击）
- 结构性趋势判断（高成本世界、AI重工业化）
- 三情景框架帮助避免单点预测的脆弱性

**失效场景**：
- AI应用端爆发速度远超预期时（好情景突然切换）
- 地缘政治黑天鹅迅速缓和时（市场会快速反应，留不够时间缩仓）
- 货币政策超预期宽松（央行突然降息/QE，托住消费端）

**与其他流派核心差异**：
- 对比"单纯AI多头"：黄哥不否认AI产业趋势，但持续提示资本开支泡沫化风险
- 对比"单纯宏观悲观派"：黄哥仍保留好情景，不做单边看空，设置情景触发机制

## 反方视角

好情景的概率可能被低估。黄哥将好情景判定为"概率最低"，但AI应用端的边际变化（如主权AI采购爆发、企业IT预算快速切换到AI）可能来得比消费端恢复更快——忽视"应用端结构性加速"可能使这套风险框架过于悲观。
