# 麦当劳 vs 蜜雪冰城：两种商业模式的对立

> 来源：260404 蜜雪冰城与麦当劳分析.md  
> 分析时间：2026-04-24 | 核心判断：两种经营哲学的本质差异，决定了它们的天花板

---

## 一、商业模式解析

### 1.1 麦当劳：地产+房东+平台

**表面现象**：卖汉堡和薯条

**实质操作**：
```
1. 麦当劳购买或长租地产
  ↓
2. 转租给加盟商（加价）
  ↓
3. 同时从加盟商销售额中抽成（3-5%）
  ↓
4. 品牌运营与品控
```

**收入结构**（黄哥的数据）：
- 地产租赁收入：大头（固定、稳定）
- 销售额分成：次要（浮动、增长）
- 品牌许可费：补充

**现金流特征**：
- 固定收入 > 浮动收入
- **风险转移**：所有经营风险都转嫁给加盟商
- 麦当劳只需要：品牌管理 + 品控 + 品牌运营

**核心哲学**：加盟商活得久 → 麦当劳收租久 → 双赢

### 1.2 蜜雪冰城：供应链+规模经济

**表面现象**：卖廉价奶茶

**实质操作**：
```
1. 极致的供应链管理（自己种果、自己做原料）
  ↓
2. 压低原料成本至极限
  ↓
3. 定价低到市场无法承受
  ↓
4. 通过规模扩张来增加利润（加盟费 + 供应链差价）
```

**收入结构**：
- 加盟费：快速回笼资金
- 供应链差价：核心利润（原料 5 元 → 加盟商 20 元）
- 规模经济：门店越多，单店单位成本越低

**现金流特征**：
- **规模驱动**：必须不断开新店才能维持利润增长
- **风险集中**：蜜雪冰城押注供应链，加盟商押注选址
- 风险转移但方向不同：麦当劳转移经营风险，蜜雪转移市场风险

**核心哲学**：卷死对手 → 市场是我的 → 加盟商死不死关我卵事

## 二、两种商业模式的对比矩阵

### 2.1 核心竞争力来源

| 维度 | 麦当劳 | 蜜雪冰城 |
|------|-------|---------|
| **护城河** | 品牌 + 地产 | 供应链成本 + 规模 |
| **可复制性** | 中等（地产贵，品牌建设慢） | 高（供应链可复制，但需巨资） |
| **抗周期性** | 强（地产租赁稳定） | 弱（取决于奶茶市场热度） |
| **利润来源** | 稳定、长期、固定 | 高增长、短期、浮动 |
| **加盟商生命周期** | 长（10+ 年） | 短（2-5 年） |

### 2.2 对加盟商的态度

**麦当劳的逻辑**：
```
加盟商活得越久 → 我收租越久 → 我赚得越多
  ↓
所以我需要：
- 更好的产品
- 更好的品控
- 更好的品牌运营
- 帮助加盟商提高盈利能力
```

**蜜雪冰城的逻辑**：
```
我有极致的供应链 → 我定价极低 → 对手都死了 → 市场是我的
  ↓
加盟商只需要：
- 交加盟费
- 从我这里买原料（我赚差价）
- 开新店扩张规模
- 至于他赚不赚钱？不重要，反正我已经收了加盟费
```

**黄哥的评价**：
> "麦当劳的价值观：加盟商活得久，我就赚得多  
> 蜜雪冰城的价值观：我只需要卷死对手，加盟商死不死关我卵事"

## 三、蜜雪冰城的致命困境

### 3.1 门店密度陷阱

**机制**：
```
蜜雪冰城开新店
  ↓
新店抢走周边旧店的客户（自相残杀）
  ↓
旧店利润下降
  ↓
加盟商回本周期拉长（原定 2 年 → 变成 3-4 年）
  ↓
加盟商现金流恶化，可能退出
  ↓
蜜雪冰城失去供应链客户（亏损）
```

**对标案例**：海底捞的困境
- 海底捞也是通过开新店来扩张
- 但新店开多了，也开始自相残杀
- 最后被迫停止扩张，甚至关闭一些门店

**蜜雪冰城的未来**：
- 现在在高速扩张（门店数在增加）
- 一旦市场饱和，会面临**单店产值下降**的反噬
- 这时候，供应链优势无法救场（因为加盟商本身就在亏损）

### 3.2 国际化的挑战

**国内为什么成功**：
- 低价 = 类刚需（刺激高频消费）
- 供应链完整（自家果园、自家工厂）
- 市场文化匹配（中国人接受 3 块钱奶茶）

**出海的困难**：
- **欧美**：饮品习惯不同（不喝奶茶），供应链成本高（无法维持低价）
- **东南亚**：有机会，因为饮食习惯接近 + 消费力偏低
- **日韩**：品质期望高，低价无法吸引（反而是高端品牌吃香）

**黄哥的判断**：蜜雪大概率在东南亚成功，但欧美失败

## 四、麦当劳的护城河

### 4.1 为什么麦当劳更稳定

**核心优势**：品牌 + 地产 + 系统化运营

```
品牌建设（50+ 年）
  ↓
全球认可的餐饮品牌
  ↓
加盟商愿意为这个品牌付高溢价
  ↓
麦当劳可以收更高的租赁费 + 更高的分成
  ↓
利润稳定且持续增长（不需要靠产品创新）
```

### 4.2 抗衰退能力

- 经济好时：人们去麦当劳吃汉堡
- 经济差时：人们去麦当劳吃最便宜的快餐
- **无论如何，麦当劳都赚钱**（因为地产租赁是固定的）

### 4.3 对比蜜雪冰城

- 蜜雪的利润依赖于：加盟商的健康 → 消费者的消费欲望
- 一旦经济衰退，奶茶是可选消费 → 需求锐减
- 加盟商倒闭 → 蜜雪冰城失去供应链客户 → 利润大幅下滑

## 五、估值对比

### 5.1 黄哥的核心判断

> "蜜雪的 PE 不应该高过麦当劳"

**理由**：

| 指标 | 麦当劳 | 蜜雪冰城 |
|------|-------|---------|
| **利润稳定性** | 高（地产租赁） | 低（供应链驱动） |
| **抗周期能力** | 强（必需品定价） | 弱（可选消费） |
| **加盟商质量** | 高（收益稳定） | 低（竞争激烈） |
| **长期增长** | 稳定（3-5%/年） | 爆发（当增速快，但短期） |
| **风险** | 低 | 高 |

**当前估值偏差**：
- 市场给蜜雪冰城的 PE 可能与麦当劳相仿或更高（因为高增长故事）
- 但从风险调整的角度，蜜雪应该有**折价**

## 六、行业发展阶段分析

### 6.1 蜜雪冰城的生命周期

```
第一阶段（2015-2020）：规模扩张期
  ✓ 高增长（50%+/年）
  ✓ 加盟商收益良好
  ✓ 市场还未饱和

第二阶段（2021-2025）：市场饱和期
  ~ 增速下降（20-30%/年）
  ~ 加盟商开始亏损（自相残杀）
  ~ 新店边际利润下降

第三阶段（2026+）：瓶颈期
  ✗ 增速进一步放缓
  ✗ 可能出现门店关闭
  ✗ 供应链优势无法救场
```

### 6.2 麦当劳的生命周期

```
第一阶段（1990-2010）：全球扩张期
  ✓ 开设新门店

第二阶段（2010-2020）：品牌优化期
  ~ 门店数稳定
  ~ 利润来自地产升值 + 租赁费上升

第三阶段（2020+）：稳定现金流期
  ✓ 地产组合优化
  ✓ 每年产生稳定现金流
  ✓ 很难破产（因为地产底线支撑）
```

## 七、对抗步骤（反方视角）

**风险 1：低估蜜雪的供应链竞争力**
- 蜜雪的供应链确实有成本优势
- 即使门店密度饱和，也可以通过提价或产品创新维持利润
- 奶茶市场本身在增长（新消费），不一定会因自相残杀而衰退

**风险 2：忽视消费心理变化**
- 年轻人对蜜雪有极强的品牌认同（性价比文化）
- 麦当劳的品牌反而在逐步老化（被认为"不够潮"）
- 黄哥的分析没有考虑代际更新

**风险 3：过度简化"加盟商福祉"与利润的关系**
- 麦当劳的高租赁费其实也在压低加盟商利润
- 麦当劳和蜜雪的区别可能只是**压榨方式不同**，而非本质差异
- 加盟模式本身就是风险转移，两家都一样

**风险 4：对中国消费升级的理解不足**
- 蜜雪可能不会永远定位在"3 块钱奶茶"
- 随着消费升级，蜜雪也可能推出高端线（类似星巴克的多品牌策略）
- 这会改变其利润结构

## 八、关键洞察与原则验证

### 原则应用：利益结构决定立场（原则 2）

**麦当劳的利益**：加盟商长期健康 → 可以收更高的租赁费

**蜜雪冰城的利益**：市场份额最大化 → 可以收更多的供应链差价

→ 两家的激励结构完全不同，导致商业哲学对立

### 结构性矛盾识别（原则 1）

**矛盾**：供应链成本优势 vs 品牌信任值的对立

- 蜜雪靠成本压低价格 → 赢得市场份额 → 但加盟商活得短
- 麦当劳靠品牌维持溢价 → 加盟商活得久 → 但入门费用高

**问题**：哪一种能走得更远？
- **短期**：蜜雪赢（规模和增长）
- **长期**：麦当劳赢（稳定和抗风险）

## 九、使用指南

### 快速判断一个快餐加盟品牌

1. **问**：这个品牌的核心竞争力是成本优势还是品牌信任？
   - 成本优势 → 风险高（容易被复制，容易自相残杀）
   - 品牌信任 → 风险相对低（难以复制，加盟商愿意长期合作）

2. **问**：加盟商的生命周期是多久？
   - 短于 3 年 → 风险高（容易离场）
   - 长于 5 年 → 风险相对低（稳定）

3. **问**：公司的利润来源是什么？
   - 加盟费 + 物料差价 → 风险高（依赖规模扩张）
   - 地产租赁 + 品牌许可 → 风险相对低（稳定现金流）

4. **问**：市场是否已经饱和？
   - 如果已经饱和，成本优势品牌会面临增速下滑 → 投资风险上升

**反方视角总结：**

黄哥的分析过度强调了加盟商的福祉与公司长期成功的关联。实际上，两种商业模式都有生存空间——蜜雪可以通过品牌升级和产品创新走向高端，麦当劳的地产护城河虽然稳定但增长有限。估值的高低应该取决于市场阶段和风险偏好，而不是简单的"PE 不应该高"。
