### STEP 2 · 决策原则提炼（6条）

**原则1：顺着物理瓶颈买标的**
- 触发条件：AI算力需求持续扩张→数据中心用电量超预期→电力基础设施成为新瓶颈
- 行动方向：发电端（天然气/光伏）→输配电端（逆变器/保护）→半导体端（SiC/GaN）→材料端（封装/晶圆气体），全链路纵向布局
- **反例边界**：若模型效率革命（如稀疏激活/量子计算）使单位算力耗电大幅下降，瓶颈转移至算法层，能源链逻辑断裂

**原则2：评级≠买卖信号，是入场优先级**
- 触发条件：同赛道多家公司并列，但评级分层（A/B/C）
- 行动方向：A级=核心仓，可以当前价直接建仓；B级=等待10-15%回调窗口；C级=观察仓，轻仓探索或等待催化
- **反例边界**：C级标的出现技术突破或重大合同，可临时升格为B级入场

**原则3：跨市场套利，锁定赛道最强链接**
- 触发条件：同一赛道在不同市场估值体系差异显著
- 行动方向：美股（技术+叙事溢价，英飞凌/MPS）→台股（供应链护城河，宏达电子）→A股（政策+产能弹性，通威/士兰微）→港股（价值洼地，华电）
- **反例边界**：台海风险升级直接影响台股标的；中美脱钩加速影响美股ADR持有安全性

**原则4：卡住"不可绕过"的材料层**
- 触发条件：识别产业链中不可替代的底层材料或组件
- 行动方向：Rogers散热基板、晶圆特种气体等属于隐形卡脖子层，具有极强定价权，优先布局
- **反例边界**：若中国完成全链路自主替代（国产化率>80%），外资材料龙头护城河削弱

**原则5：天然气≠传统能源，是AI时代的过渡基础设施**
- 触发条件：可再生能源供给不稳定→天然气作为调峰电源地位短期无法替代
- 行动方向：EQT/Expand Energy不是"旧能源"逻辑，而是AI算力基础设施链的上游供应商
- **反例边界**：电池储能技术突破（固态电池大规模量产）可削弱天然气调峰作用

**原则6：光伏链优选行业出清后的剩者**
- 触发条件：光伏行业经历惨烈价格战和产能出清周期
- 行动方向：通威股份（硅料+电池片双龙头）+西子洁能（运营商）锁定出清后的胜者
- **反例边界**：新一轮产能扩张（东南亚产能转移后仍过剩），竞争格局二次恶化

### STEP 3 · 因果链建模（三场景）

**场景A：AI算力牛市（主场景）**
```
[数据中心GPU订单持续超预期] 
→ [权重：高] 
→ [美国电力缺口扩大至150GW，天然气发电+光伏储能同步受益] 
→ 核心仓：EQT/通威/宏达电子/MPS（A级），分批建仓
→ 观察仓：英飞凌/固德威（等催化）
```

**场景B：电动化加速（协同场景）**
```
[800V平台EV车型全面普及，2026年渗透率>40%] 
→ [权重：高] 
→ [SiC MOSFET需求缺口→英飞凌/STM/士兰微量价齐升] 
→ 布局功率半导体，优先选有SiC产能的龙头
```

**场景C：宏观流动性收紧（防御场景）**
```
[美联储加息预期反弹/中国信用扩张放缓] 
→ [权重：中] 
→ [成长股估值承压，A/B级标的回调至目标价] 
→ 持现等待，B/C级标的绝不追高；C级华电港股作为防守仓备选
```

### STEP 4 · 认知偏好画像

**风险偏好：情境切换型**
- A级标的：接受5-10%短期回撤，长期持有，不因波动止损
- B级标的：设定入场价格，低于目标价才行动
- C级标的：轻仓探索，严格控制在总仓5%以内

**时间维度权重**：结构性赛道（70%）× 短期催化（30%）
- 能源链是3-5年结构性机会，但政策窗口（美国基建法案、中国新能源电价政策）可带来超额收益

**决策锚点优先级**（从高到低）：
1. 产业链逻辑（物理瓶颈在哪）
2. 财务质量（经营现金流是否真实）
3. 市场估值（作为入场时机判断，非核心变量）
4. 市场情绪（最后参考因素）

**认知盲区**：
- 黄哥的体系对**地缘政治突变风险**（台海→宏达电子、中美脱钩→美股ADR）建模相对不足，跨市场布局存在隐性集中度风险
- **利率敏感性不对称**：美/港/台/A四市对利率反应机制不同，整体组合的久期风险未充分显性化

## 五、反方视角（对抗步骤）

**这套框架最可能错在哪里**：

假设错误——AI数据中心用电需求是「确定性瓶颈」，但事实上，**Nvidia在Rubin/Blackwell架构中已将能耗降低40%**，若2026-2027年AI模型效率提升速度远超市场预期，「AI→用电→能源链」的正向传导链可能比预期弱得多。结果是能源股获得了叙事溢价但基本面并未同步兑现，L1-L4层标的面临戴维斯双杀风险。
