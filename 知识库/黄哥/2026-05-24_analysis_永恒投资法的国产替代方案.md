## 上下文：什么是"永恒投资法"

Harry Browne 1981年提出的 **Permanent Portfolio**：股票25% + 长期国债25% + 黄金25% + 现金25%。核心理念——四种资产对应四种宏观状态：

- **股票** ← 经济繁荣
- **长债** ← 通缩衰退
- **黄金** ← 通胀失控
- **现金** ← 流动性紧缩

四种状态总会有一个发生，因此组合永远有一部分在涨。**国产替代方案**就是用 A股宽基（沪深300）+ 国债ETF + 黄金ETF + 货币基金 来还原这套思想。

## STEP 2 · 决策原则提炼（5条）

### 原则1：**用"夏普比率"而非"绝对收益"做评判**

- **触发条件**：评估任何资产配置方案。
- **行动方向**：把"年化收益÷波动率"作为首要指标，而非看哪个数字更大。
- **反例边界**：当投资期限极长（>20年）且投资者具备情绪韧性（不会在-40%时割肉）时，纯股票方案的复利可能反超组合方案——但前提是"扛得住"。

### 原则2：**最大回撤是隐藏的"出局概率"**

- **触发条件**：选择长期持有的底层资产。
- **行动方向**：把-42%这种数字翻译为"普通人有多大概率在底部割肉退出"——经验上接近 70%。
- **反例边界**：仅当投资者已经经历过完整熊市训练（如2018或2022），才有资格在底部加仓。

### 原则3：**资产去相关，胜过押注单一明星**

- **触发条件**：在多个资产中选择。
- **行动方向**：宁可降低期望收益，也要让资产之间的相关性接近零或负——这是组合工程的本质。
- **反例边界**：在所有资产同步暴跌的"系统性流动性危机"中（如2008年3月、2026年初），相关性会瞬间收敛到1，组合保护失效。这时只有现金和黄金的部分保护起作用。

### 原则4：**黄金不是投机标的，是组合的"必备模块"**

- **触发条件**：构建跨周期组合。
- **行动方向**：哪怕黄金本身波动率13.56%、回撤20.5%（看似很"激进"），它依然是组合中的稳定器——因为它在股票崩盘时往往上涨。
- **反例边界**：在强美元+实际利率上行的周期（2013-2015），黄金会拖累组合，但配置比例不应因此调整——这就是"永恒"二字的含义。

### 原则5：**"国产替代"的本质是承认本土资本市场的特性**

- **触发条件**：把海外经典策略本土化。
- **行动方向**：用国债ETF替代美债（流动性低、久期短），用沪深300替代标普（更高波动、更低分红），用境内黄金ETF替代实物金（汇率风险被吸收）。结构等价，但每个组件的特征都被本土市场扭曲过。
- **反例边界**：当人民币长期单边贬值时，未对冲汇率的境内黄金ETF会双重受益（金价涨+人民币贬），夏普比率会被高估，需要剥离汇率因素后再评估。

## STEP 3 · 因果链建模（升级版）

### 路径A：组合在熊市中的运作

```
[沪深300 -40%信号] →
[权重评估：股票25% × -40% = -10%组合冲击] →
[叠加判断：黄金通常上涨 +15% × 25% = +3.75%，国债 +3% × 25% = +0.75%] →
[最终影响：净冲击约 -5.5%] →
[行动：rebalance——卖出涨幅资产，买入跌幅资产]
```

### 路径B：组合在通胀周期的运作

```
[CPI > 4%信号] →
[权重评估：黄金强势(+20%)，债券压力(-5%)，股票分化] →
[叠加判断：黄金贡献+5%，债券拖累-1.25%，股票±0] →
[组合总收益约 +4%，跑赢现金、跑赢债券] →
[行动：维持原比例，让黄金做主引擎]
```

### 路径C：组合在流动性危机中的运作

```
[所有资产同步暴跌信号] →
[权重评估：常规相关性失效，黄金可能也跌] →
[叠加判断：仅现金/货币基金保护起作用 = 25%缓冲] →
[组合冲击：-15% ~ -20%（小于沪深300的-40%）] →
[行动：等待，不抄底、不调仓——这是"永恒"哲学的考验时刻]
```

## STEP 4 · 认知偏好画像

### 风险偏好
**情境切换型 — 偏保守**
- 在"无法判断宏观方向"的状态下，默认采取**风险平价**的均衡分配；
- 不押单一观点（不会全仓黄金、不会全仓股票），即使知道某资产明显占优；
- **切换条件**：仅在出现明确的"系统级信号"（如美债违约、严重通胀失控）时，才会偏移基准比例 ±5%。

### 时间维度
- **长期结构 > 短期信号**：5%vs15%的差异要在5-10年才能体现复利价值；
- 评估周期至少是一个完整经济周期（10年），短于此周期的回测都是噪音。

### 决策锚点（优先级）
1. **数据驱动**：年化、波动、回撤这三项硬指标，不可替代
2. **结构驱动**：四资产分类的因果逻辑（繁荣/衰退/通胀/紧缩）
3. **情景验证**：在多个历史时段（2008/2015/2020）回测组合表现
4. **叙事驱动**：最弱，只在跟散户解释时使用

### 认知盲区
- **未对冲汇率风险**：境内黄金ETF嵌入人民币贬值收益，未来若人民币转强，黄金收益将被压缩
- **未纳入海外资产**：组合完全本土化，缺少美股/海外REITs等对冲国内系统性风险的模块
- **未考虑税收和摩擦成本**：rebalance的交易成本、税费会侵蚀组合的夏普优势
- **回测期偏差**：如果回测周期主要在2015-2025黄金大牛市，会高估黄金的稳定贡献

## STEP 5 · 表达风格建模

### 句式偏好（基于图表+黄哥过往风格）
- **数据先行，论述滞后**：先给出年化/波动/回撤三大指标，再让读者自己推导含义
- **极简对比格式**：表格而非段落，五行能说清楚的事不写五段
- **不解释方法论**：默认读者已经懂"永恒投资法"，不科普背景

### 是否使用绝对化表达
- 用"+15.32%"这种精确小数，而非"约15%"——**数据精确化**是黄哥的偏好
- 但**不会**说"这就是最优组合"——只给数据，让数据自己说话

### 类比偏好
- 偏好**金融工程类比**（夏普比率、风险平价、相关性）
- 不偏好**情绪化类比**（不会说"避风港""定海神针"）

### 情绪基调
**冷静分析型 + 隐含警示**
- 表面给数据，但反差数据本身就是警示：你以为沪深300回报4.76%还不错？看看-42%回撤
- 不直接说"别买沪深300"，但数据已经说完了

### 仿写校验标准（黄哥"不会写"的5种句式）
1. ❌ "这套组合稳赚不赔，建议人手一份"（绝对化推销）
2. ❌ "财富自由从此开始"（情绪营销话术）
3. ❌ "适合所有年龄段的投资者"（笼统模糊）
4. ❌ "黄金一定会涨，国债一定会跌"（单向预测）
5. ❌ "本组合超越巴菲特的伯克希尔"（夸大比较）

如果一篇仿写文里出现以上句式，就不是黄哥的逻辑。

## STEP 6 · 输出格式

### 一、认知核心原则（可迁移）

**"用组合工程降低出局概率"** 适用于一切多变量决策场景：
- 创业：不要all-in一个赛道，分散到三个去相关的方向
- 职业：不要只押公司岗位，配置技能/人脉/资本三个维度
- 学习：不要只读一个流派，跨学科的知识相关性低于0.5才有阿尔法

### 二、决策流程模型

```
任意决策场景
    ↓
是否能预测未来？
    ↓ 否
是否能识别"必然发生但不知何时"的多种情境？
    ↓ 是
为每种情境分配"独立的押注模块"
    ↓
每个模块权重相等（除非有强证据）
    ↓
定期rebalance（让涨的卖一些，跌的买一些）
    ↓
评估指标：不是"最高收益"，是"风险调整后收益"
```

### 三、认知偏好画像

风险偏好：情境切换型偏保守 / 时间维度：长期 / 决策锚点：数据>结构>情景>叙事 / 盲区：未对冲汇率、未纳入海外、未考虑摩擦成本

### 四、表达风格建模

冷静、精确、数据先行、隐含警示、拒绝绝对化承诺。

### 五、模型可靠性评估

**优势场景**：
- 通胀+加息周期（黄金扛旗，股债同跌时组合靠黄金维持）
- 宏观不确定性高、单一观点无法验证的窗口期（如政策转向期）
- 长周期投资（>10年），让复利在低波动基础上累积

**失效场景**：
- 长牛市单边上涨（如2017美股、2007A股），纯股票会跑赢组合至少3-5倍
- 系统性流动性危机（如2026/Q1），所有资产同步暴跌，去相关失效
- 黄金长期熊市（如2013-2015年），组合最大引擎熄火，年化可能降至3%以下

**与其他流派的核心差异**：
- vs **巴菲特/伯克希尔**：巴菲特是"集中押注+护城河"，永恒投资法是"分散押注+宏观状态"——一个赌人，一个赌制度
- vs **桥水全天候**：桥水加杠杆做风险平价，永恒投资法不加杠杆——更保守、更适合个人

## 反方视角（强制对抗步骤）

**这个组合最可能错在哪里**：

1. **回测窗口的幸存者偏差**：5.93%的年化和-8.81%的回撤，依赖于"过去这段时间里黄金没有10年熊市、债券没有违约、股票没有连续负回报"。若未来出现日本式失去的30年（股票横盘+利率为零+黄金无趋势），这个组合的年化可能滑落到 2%以下，跑不赢通胀。

2. **rebalance的隐含假设**：rebalance的核心收益来自"涨的资产会回归均值，跌的资产也会回归"。但如果某类资产进入**结构性死亡**（如90年代日本股票、2015-2018年部分REITs），rebalance就是"持续往坟墓里加钱"。

3. **货币基金的0%年化是个警钟**：在零利率时代，组合的25%现金部分将完全不产生收益，整体年化会被强制下调1-1.5pp。如果中国进入零利率周期，国产替代方案的5.93%年化会下滑到 4.5%以下，那时夏普优势就被大幅削弱。

## 与已有体系的交叉验证

**与 [[2026-05-19_framework_投资1.0模型显性化与估值方法论]]**：
- 投资1.0模型强调"期权价值归零"的保守估值
- 永恒投资法是这种保守哲学的**组合层落地**——不预测、不押注、用结构性分散对冲未知

**与 [[2026-05-22_decision_失败投资的四条反脆弱原则]]**：
- 失败投资经验中的"少决策、大押注、快纠错"是单标的反脆弱
- 永恒投资法是**组合反脆弱**——通过资产分散，让"快纠错"自动化（rebalance机制）

**与 [[2026-04-25_framework_物理瓶颈论]]**：
- 物理瓶颈论说"任何赛道都有极限"
- 永恒投资法说"任何单一资产都有失效期"——同源思维
