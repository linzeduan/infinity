# 研究公司基本面的核心问题框架

来源：黄哥投资决策体系  
分析时间：2026-04-27 | 核心判断：用4层递进问题 + 逆向思考，可快速识别公司的真实价值和风险边界

---

## 【第一层】利益结构（钱从哪里来）

**核心问题**：这家公司的现金流来自谁，依赖什么条件持续存在？

**深度追问**：
1. 公司的主要收入来源TOP 3是什么？占比各多少？
2. 最终的付费方是谁？这些付费方的支付能力是否稳定？
3. 这笔钱依赖的经济激励或制度条件是什么？
4. 如果这个条件变了（政策/竞争/需求），收入模式会崩溃吗？

**数据追踪指标**：
- 收入来源集中度（单一客户占比、单一产品占比）
- 用户/客户的续费率或复购率
- 现金流与收入的差距（是否有大量应收账款）
- 最大客户流失风险是什么

**黄哥案例**：
- 茅台：钱来自"权力消费"而非酒的品质 → 如果权力结构变化，溢价瞬间瓦解
- 特斯拉：未来的钱来自FSD订阅（软件），而非卖车硬件

---

## 【第二层】结构矛盾（行业根本问题）

**核心问题**：这个行业有什么根本性的、无法被政策解决的矛盾？

**深度追问**：
1. 这个行业的需求方和供给方之间存在什么失衡？
2. 是什么因素导致这种失衡——周期性、结构性还是制度性？
3. 如果这个矛盾存在，公司即使做得再好也无法突破什么边界？
4. 这个矛盾会随时间自我修复还是逐渐恶化？

**数据追踪指标**：
- 行业增长率是否在放缓（结构性衰退信号）
- 新进入者数量和退出者数量的对比
- 头部企业的利润率趋势
- 行业的杠杆率和现金流压力

**黄哥案例**：
- 养老产业：需求存在但支付能力永远缺失 → 注定烧钱，政策补贴也救不了
- 房地产：人口红利尾部已现 → 需求断崖，不是政策能解决的
- 重资产生意：现金周转慢 + 融资成本上升 → 经济下行时风险倍增

---

## 【第三层】时间节点（拐点预判）

**核心问题**：这个公司和行业最关键的拐点在什么时候，怎么追踪？

**深度追问**：
1. 这个行业/公司的核心数据指标是什么？（不是财务指标，是行业的"晴雨表"）
2. 这个指标的历史周期是多长？（3年？10年？）
3. 当前这个指标处于周期的哪个位置？
4. 根据历史周期，下一个拐点会在什么时候出现？

**数据追踪指标**：
- 领先指标（比财务报表早出现的信号）：用户增长、产能利用率、库存、订单量
- 同步指标：收入、利润、市场占有率
- 滞后指标：股价、整合度

**黄哥案例**：
- 生猪周期：追踪"繁育母猪去产能速度" → 可提前6个月预测反弹时间
- 房价顶峰(2018-19)：对标"第三次婴儿潮出生数量尾部(2017年)" → 需求消退信号
- 特斯拉FSD：追踪激活用户数 → 判断订阅收入的爆发点

---

## 【第四层】失效条件（风险边界）

**核心问题**：在什么条件下，这个投资逻辑会彻底崩塌？

**深度追问**：
1. 如果[最可能发生的坏事件]发生，这个公司还值钱吗？
2. 核心竞争力有多容易被复制或替代？
3. 政策、竞争、技术哪个因素的影响最致命？
4. 在最坏情况下，这个公司会沦为什么（清退、被收购、行业底部）？

**数据追踪指标**：
- 公司对单一政策/客户/产品的依赖度
- 竞争对手进入的难度
- 管理层的风险敏感度（是否主动规避风险）
- 现金储备和融资能力

**黄哥案例**：
- 特斯拉：政策收紧 + 融资环境恶化 + 机器人成本无法下降 → "市梦率"瞬间破灭
- 苹果：如果医疗AI的过程数据被法律禁用 → 整个布局失效
- 互联网生意：平台算法变化 → 业务瞬间崩塌（如抖音美食博主的命运）

---

## 【额外追问】逆向思考法（黄哥常用）

**问题**：如果你想让这个公司彻底变糟，最有效的做法是什么？

这个反向问题可以帮你快速识别：
- 公司最脆弱的环节
- 竞争对手最有可能下手的地方
- 哪些风险是"最不可控"的

**例子**：
- 茅台变糟的最有效方式 → 改变权力流向结构（如反腐深化）
- 特斯拉变糟的最有效方式 → 推迟FSD全面放开的时间窗口
- 养老公司变糟的最有效方式 → 老龄化进程放缓（几乎不可能，所以这个行业本质烧钱）

---

## 【执行顺序】研究一家新公司的标准流程

1. **第一轮：40分钟快速扫描**
   - 第一层：钱从哪里来？
   - 第二层：行业结构矛盾是什么？
   - 第四层：最大的风险是什么？
   - → 输出：这家公司值不值得深入研究

2. **第二轮：2小时深度研究（如果通过第一轮）**
   - 第三层：追踪关键数据指标，确认周期位置
   - 补充行业对标分析（竞品相同问题的答案）
   - → 输出：买不买，买多少，什么时候卖

3. **第三轮：持续监测（已买入）**
   - 每个月追踪第三层的关键指标
   - 每个季度重新评估第二、四层的风险
   - 设置止损线：在什么条件下一定要卖

## 反方视角

这套框架的局限性：
1. **假设参与者理性** → 在极限政策环境（战争、政变）下，人们会做出"经济上非理性但政治上理性"的决策，此框架失效
2. **对前沿技术成本曲线拐点判断有限** → 需要硬技术专家补充（如芯片、机器人的实际成本下降曲线）
3. **数据充分的领域最有效** → 对信息不透明的私企、初创公司效果有限
4. **样本偏差** → 框架基于黄哥4月的分析，市场环境如果剧变，历史周期参考可能失效
