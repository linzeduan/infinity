## 二、决策原则提炼

**原则1：政策确定性 → 持仓不动**
- 触发信号：政策从地方试点升级为国家战略（2会写入工作报告）
- 判断逻辑：政策确定性意味着需求端有保障，产业链上下游都会受益
- 对应操作建议：持有电网设备等受益标的，不因短期波动而止损
- 反例边界：若政策出现反转信号（如"暂停"、"调整"），则需要重新评估

**原则2：订单放量 + 财报超预期 → 业绩兑现期确认**
- 触发信号：华为昇腾订单排至2027年，各大厂财报超预期
- 判断逻辑：从概念炒作转向产业逻辑定价，业绩成为估值支撑
- 对应操作建议：可逐步加仓或继续持有，但不追涨
- 反例边界：若订单增速放缓或财报低于预期，则需要减仓

**原则3：大涨后逻辑依然很硬 → 等待合适机会，不追涨**
- 触发信号：板块大涨，但基本面逻辑未改变
- 判断逻辑：追涨容易踩高点，等待回调或整理后的机会更优
- 对应操作建议：观望，不追涨
- 反例边界：若出现新的催化（如重大订单、政策突发），则可考虑参与

**原则4：新方向纳入观察 → 等待确定性信号**
- 触发信号：稀有金属、化工等新方向进入视野
- 判断逻辑：国产算力需要进一步观察，新方向更需要等待确定性
- 对应操作建议：观察，不操作
- 反例边界：若新方向出现明确的产业链逻辑或政策支持，则可纳入

## 三、持仓管理逻辑

**持仓标的与态度：**

| 标的          | 态度    | 理由                   |
| ----------- | ----- | -------------------- |
| 电网设备        | 持仓不动  | AI数据中心用电需求爆发，政策与需求共振 |
| 国产算力（昇腾产业链） | 观察+等待 | 逻辑硬但已大涨，等待合适机会       |
| 电池          | 暂时观望  | 无新变化                 |
| 券商          | 暂时观望  | 无新变化                 |
| 恒生科技        | 暂时观望  | 无新变化                 |
| 军工龙头        | 暂时观望  | 无新变化                 |
| 港股创新药       | 暂时观望  | 无新变化                 |
| 稀有金属、化工     | 纳入观察  | 新方向，等待确定性            |

**持仓管理原则：**
- 对于政策确定性强的标的（电网设备），不因短期波动而止损
- 对于业绩兑现期的标的（国产算力），等待合适机会，不追涨
- 对于新方向，先观察，等待确定性信号后再操作

## 四、风险意识画像

**对哪类风险最敏感：**
- 政策风险：若"算电协同"政策出现反转，电网设备等标的会大幅回调
- 业绩风险：若华为昇腾订单增速放缓或财报低于预期，国产算力板块会面临估值压力
- 追涨风险：大涨后追涨容易踩高点，需要等待回调

**规避风险的方式：**
- 对于政策确定性强的标的，持仓不动，相信政策的长期支撑
- 对于已大涨的标的，不追涨，等待合适机会
- 对于新方向，先观察，等待确定性信号

**持有期望：**
- 电网设备：中长期持有，受益于"算电协同"国家战略
- 国产算力：中期持有，等待业绩兑现期的估值提升
- 其他标的：观察期，等待新的催化

## 五、表达风格建模

**用词习惯：**
- 使用"大浪淘沙，沉者为金"作为品牌标签
- 频繁使用"机会"、"催化"、"爆发"等市场黑话
- 强调"不会频繁操作"、"等风来"的投资哲学

**确定性程度：**
- 对政策确定性的表述最强（"首次被写入工作报告"、"明确为新基建工程"）
- 对已发生的事实表述确定（订单数据、财报超预期）
- 对未来方向表述为"观察"或"纳入进来"，保留灵活性

**对比频率：**
- 本篇未进行前日观点回顾，但强调了"不赘述"其他方向的无新变化
- 隐含对比：大涨后的国产算力 vs 无新变化的其他标的

**风险提示：**
- 明确声明"暂时不追涨"，表明对追涨风险的认知
- 强调"等待合适机会"，体现风险管理意识

## 六、模型可靠性评估

**此思维在什么市场阶段最有效：**
- 政策驱动的产业升级阶段（如"算电协同"从试点到国家战略）
- 业绩兑现期（订单放量、财报超预期）
- 板块大涨后的整理期（逻辑硬但需要等待回调）

**在什么情况下会失效：**
- 政策突然反转（如"暂停"、"调整"）
- 业绩低于预期（订单增速放缓、财报不达预期）
- 黑天鹅事件（地缘冲突、产业链断裂）
- 市场情绪极度悲观或乐观时，基本面逻辑被情绪压倒

**与其他投研体系的差异：**
- 相比纯技术面分析：更重视政策与产业逻辑的确定性
- 相比纯基本面分析：更强调时机选择（不追涨，等待合适机会）
- 相比短线操作：强调"不频繁操作"，追求确定性而非频率

## 七、对抗步骤

**反方视角：** 这个判断最可能错在哪里？

1. **政策执行风险被低估**：虽然"算电协同"写入工作报告，但地方执行力度、投资进度可能不及预期，导致需求端释放缓慢
2. **国产芯片技术路线风险**：华为昇腾虽然订单充足，但与国际先进芯片的性能差距仍存，若应用端出现"不够用"的反馈，可能面临需求调整
3. **估值透支风险**：国产算力已大涨，当前估值可能已充分反映了业绩兑现预期，后续上升空间有限
4. **电网设备的需求假设**：AI数据中心用电需求爆发是基于"大模型训练规模持续扩大"的假设，若大模型应用增速放缓，需求可能不及预期

## 八、使用指南

**适用场景：**
- 判断政策驱动的产业升级机会
- 评估业绩兑现期的持仓策略
- 决定是否追涨已大涨的板块

**不适用场景：**
- 短线交易（此思维强调"不频繁操作"）
- 政策不确定的新兴产业
- 基本面出现明显恶化的标的

**迭代方向：**
- 持续跟踪华为昇腾订单进度与财报兑现情况
- 监测"算电协同"政策的地方执行进度
