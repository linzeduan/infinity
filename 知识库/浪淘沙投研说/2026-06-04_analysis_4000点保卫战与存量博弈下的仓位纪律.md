## STEP 2 · 决策原则提炼

### 原则①：连续3日跌停家数减少 → 小仓位博反弹

- **触发信号**：跌停家数连续3日下降
- **判断逻辑**：恐慌情绪退潮的技术标志，短期反弹概率升高
- **对应操作**：大跌后小仓位入场博反弹，非重仓押方向
- **反例边界**：若跌停减少只因交易日效应（如节前效应），而非真实情绪修复，则信号失效；需同时验证成交量是否温和放大

### 原则②：新开户暴增 + ETF流出 = 中期风险旗

- **触发信号**：单月新开户276万+（同比+78%），叠加宽基ETF年初大幅流出
- **判断逻辑**：散户涌入是历史见顶配置，"新韭菜赶上了赚钱的机会"=真正的顶部叙事
- **对应操作**：整体降仓位，保留子弹应对深度调整
- **反例边界**：若新开户激增同时伴随国家队净买入（如救市操作），顶部信号会被对冲，减仓逻辑需重新评估

### 原则③：科技高位出货特征明确 → 不抄底，不追涨

- **触发信号**：年初宽基ETF大幅流出 + 科技板块剧烈震荡 + 高低切换频繁
- **判断逻辑**：三重信号叠加="先拉后砸"出货模式，而非正常调整
- **对应操作**：科技仓位不加，等待充分消化后再评估
- **反例边界**：若科技板块内部出现政策层面强催化（如AI补贴政策落地），需临时切换逻辑

### 原则④：基本面验证中的品种，杀跌不出

- **触发信号**：商业航天/军工连续杀跌，但海外利好持续兑现
- **判断逻辑**：价格下跌≠价值下降，大A对海外利好反应迟钝是A股结构特征，不是趋势否定
- **对应操作**：继续持有，耐心等待市场反应
- **反例边界**：若连续杀跌同时出现政策层面利空（如预算削减、采购推迟），则需重新评估持有逻辑

## STEP 3 · 持仓管理逻辑

```
科技板块（高位+出货特征+60%加息风险）
  → 不确定性高 → 保持现仓，不追涨，不加仓
  → 大跌时可小仓位博反弹，但控制比例

电池方向（新高后震荡调整完成）
  → 信号积极 → 可以考虑布局，储能持续看好
  → 避开散户扎堆的锂矿（博弈大）

商业航天/军工（连续杀跌但逻辑未变）
  → 继续持有，不因短期跌幅离场

电网设备/恒生科技/券商/港股创新药
  → 无新增信息 → 维持现状，暂时观望

整体仓位
  → 降仓+留子弹 是当前最优策略
  → 等待4000点或以下的买入机会
```

## STEP 4 · 风险意识画像

**最敏感的三类风险**（按敏感度排序）：

1. **流动性收紧风险**：ADP超预期→加息概率60%→这是全文最重要的宏观背景，每次提到科技都隐含"利率是科技估值的天花板"
2. **聪明资金出货风险**：ETF流出+高低切换="筹码转移给散户"，本质是流动性陷阱
3. **散户羊群效应风险**：新开户暴增是提醒散户读者反向思考的信号，而非跟随入场

**规避风险的主要工具**：
- 第一工具：降仓位（多次强调"保留子弹"）
- 第二工具：结构性转换（高位科技→低位电力/红利）
- 第三工具：纪律优先于判断（"不乱动"是核心纪律）

**持有期望**：
- 短期（博反弹）：大跌后小仓位入场，博3-5天反弹
- 中期（H2布局）：等指数到3800-4000区间建立底仓，目标4400-4500
