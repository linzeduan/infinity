## 二、交易决策核心原则

### 原则1：高位不追，等回调消化估值
- **触发信号**：热门赛道短期涨幅过大、交易拥挤、场内溢价高
- **判断逻辑**：情绪+资金抱团推动的上涨，一旦抱团松动即深度回调；哪怕长期逻辑硬，也必须等估值消化
- **操作建议**：持币观望，等待回调至合理区间再考虑介入
- **反例边界**：当回调幅度已充分（如30%+）且基本面逻辑未破坏时，不适用"继续等"——此时应分批建仓

### 原则2：区分"机构撤退"与"市场崩盘"
- **触发信号**：宽基ETF持续流出 vs 指数走势背离
- **判断逻辑**：宽基流出≠大盘走熊，可能是风格切换中的资金搬家（从蓝筹到成长），本质是结构性行情而非系统性风险
- **操作建议**：不因宽基流出而恐慌抛售持仓，关注资金去向而非资金流出本身
- **反例边界**：当宽基流出+指数破位+成交量萎缩三者同时出现时，"风格切换"的解释失效，应转为防御

### 原则3：跌下来是机会，但加仓要有节奏
- **触发信号**：连续大跌后出现企稳迹象
- **判断逻辑**：牛市急刹车的目的是洗盘而非转势，跌出空间=跌出机会；但"频繁加仓成重仓"是最大忌讳
- **操作建议**：分批次、有间隔地建仓，每份仓位独立决策，不因"已经跌了很多"而一次性重仓
- **反例边界**：当下跌伴随基本面恶化（如业绩暴雷、政策转向），"跌出机会"的逻辑不成立——此时应止损而非加仓

### 原则4：强势板块用"移动止盈"而非"目标价止盈"
- **触发信号**：持仓方向因消息面刺激继续上涨
- **判断逻辑**：强势不言顶，设定固定目标价会过早离场；但必须预设"如果转弱怎么办"的退出条件
- **操作建议**：上涨继续拿，如果开始连续下跌则止盈一半——既保留上行空间，又锁定部分利润
- **反例边界**：当板块估值已到历史极端值（如PE>99%分位），即使趋势未破也应主动减仓

### 原则5：被市场整体拖累的板块，不做被动决策
- **触发信号**：个股/板块有独立利好或异动，但被大盘系统性下跌拖累
- **判断逻辑**："泥沙俱下"时信号失真，此时买卖决策容易被情绪主导
- **操作建议**：先等着，等市场情绪稳定后再按独立逻辑判断
- **反例边界**：如果泥沙俱下持续时间超过2-3周且板块毫无反弹动力，说明可能不是"被拖累"而是自身逻辑已弱化


## 三、实盘操作流程：当前市场状态下的决策树

```
市场大跌 → 
├─ 持仓方向是否有利好消息支撑？
│   ├─ 是（如电池/储能：中美会议点名+产业合作）
│   │   └─ 继续持有；若连续下跌→止盈一半
│   └─ 否
│       └─ 等待，不加仓也不割肉
├─ 是否有现金可用于建仓？
│   ├─ 是 → 目标方向是否已充分调整？
│   │   ├─ 是（如电网设备连跌两天）→ 可建仓一份，但禁止频繁加至重仓
│   │   └─ 否 → 继续等
│   └─ 否 → 保持现有仓位，不做变动
└─ 是否出现"最后一杀"确认信号？
    ├─ 已确认 → 准备分批布局
    └─ 未确认（当前状态）→ 边走边看，保持耐心
```


## 四、持仓管理画像

### 当前态度矩阵

| 方向 | 态度 | 触发条件 | 下一步 |
|------|------|----------|--------|
| 电池/储能 | **持有，强势中** | 阳光电源合作+中美会议点名 | 继续拿；若连续跌则止盈一半 |
| 电网设备 | **可建仓一份** | 连续两天大跌 | 新仓可进，但绝不要频繁加仓成重仓 |
| 军工/商业航天 | **加仓第三份** | 跌出机会 | 可进第三份，后续考虑做T |
| 港股创新药 | **等待** | 有异动但被大盘拖累 | 泥沙俱下，先等着 |
| 证券 | **等待** | 被大盘拖累 | 先等着 |
| 恒生科技 | **等待** | 被大盘拖累 | 先等着 |
| 化工/稀有金属/有色 | **观察** | 尚未建仓 | 继续观察 |

### 仓位管理特征
- **加仓纪律**：单方向最多3份（如军工已进第三份），之后需要做T而非继续加
- **止盈纪律**：强势方向用"连续下跌"触发止盈，而非预设目标价
- **等待纪律**：不明确的方向宁愿不动——持仓管理中"不动"是最被低估的操作


## 四、风险管理哲学

### 风险敏感度排序
1. **情绪/拥挤度风险**（最高敏感级）：散户疯狂上杠杆+交易拥挤+场内高溢价 = 最危险的组合
2. **技术面破位风险**：穿头破脚等顶部形态出现后，技术派资金离场形成自我实现的下跌
3. **外部传导风险**：美股AI泡沫预警可能通过情绪传导至A股科技线
4. **政策/宏观风险**（当前敏感度较低）：中美缓和信号反而提供利好支撑

### 风控方式
- **不追高**：短期涨幅过大的热门赛道绝对不追
- **分批建仓**：跌下来不一次性重仓，而是分份进入
- **移动止盈**：强势方向用趋势跟踪止盈，而非预设目标
- **等待优先**：不明确时选择等待，将"不动"视为主动决策而非被动犹豫

### 持有期望
- **短期为主**：日评级别操作，关注的是"明天开盘怎么做"
- **趋势跟随**：短期操作但顺应中期趋势（震荡上行），不逆势
- **快速反应**：一旦信号转变（如连续下跌→止盈），立即执行

## 六、模型可靠性评估

### 优势场景
- **震荡市中表现最优**：在"非熊非牛、结构性分化"的环境中，其"分批建仓+不追高+等待优先"的策略能有效控制回撤并捕捉轮动机会
- **技术面与资金面共振时**：当K线信号（如穿头破脚）与资金行为（机构减持）同向时，其判断准确率较高
- **政策驱动行情**：能快速识别政策信号（如中美会议点名）并映射到具体板块

### 失效场景
- **极端单边牛市**：在普涨行情中，"不追高""等回调"会系统性踏空——因为强势板块可能不给出像样的回调
- **流动性危机/系统性崩盘**：当市场从"结构性调整"演变为"系统性风险"时，"跌出机会"的判断会过早抄底——泥沙俱下之后可能还有更深的下杀
- **长期横盘**：在无趋势的长期震荡中，频繁的"建仓一份""止盈一半"操作会累积交易成本而收效甚微

### 与黄哥体系的核心差异
浪淘沙关注的是"这一周怎么操作"，黄哥关注的是"这个时代在发生什么"——前者是战术层面的快速反应系统，后者是战略层面的认知框架。两者互补：黄哥定方向（长期哪些赛道值得关注），浪淘沙定节奏（现在该不该进场）。
