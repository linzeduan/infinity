# 浪淘沙快速决策系统 · 综合模型 v1.0

> **刷新状态**：创建于 2026-06-10（吸收 29 篇日评分析，2026-04-26 ～ 06-08）
> **下次刷新触发**：再累计 10 篇，或其核心框架失效（如"国家队托底"逻辑被打破）时立即刷新
> 定位：短周期（日-周）实盘操作系统。与黄哥模型互补——黄哥管"该不该买、值多少钱"，浪淘沙管"今天加不加、何时撤"。

## 〇、系统一句话

**在承认"无法预测明天"的前提下，靠仓位纪律、拥挤度监测和"留子弹等错杀"来挣确定性的钱**——他的优势不在预测准，而在每天都给出"明天开盘怎么做"的可执行界面。

## 一、交易决策核心原则（10 条规则集）

### 原则 1：无新增信息时，不操作本身就是最优决策
- 触发：市场剧烈波动但基本面无新变化
- 逻辑：价格波动 ≠ 投资逻辑改变；高波动+信息空白=操作代价最高的窗口，频繁操作只会亏更多
- 反例边界：量价配合的明确突破信号出现时，果断行动（05-27 当天全文不足 200 字，"什么都不做"就是建议本身）

### 原则 2：拥挤度是系统性风险的先行指标
- 触发：资金全面集中单一方向（"资金全部在科技上"）；散户涌入数据（新开户 +78% = 韭菜入场见顶信号，06-04）
- 逻辑：极度拥挤时任何利空都会引发踩踏；"不存在高低切，只剩科技内部高高切"是危险状态（05-25）
- 操作：不追加拥挤板块，降仓而非加仓（高波动下机构的行为规律，05-26）
- 反例边界：板块有超预期催化剂时，拥挤本身不是减仓理由

### 原则 3：区分"价值错杀"与"趋势破坏"——量能是判别器
- 触发：看好的板块跟随大盘下跌
- 逻辑："只是被市场带下去"→持有甚至加仓；下跌伴随成交量异常放大（机构出货）→重新评估
- 实例：3.5 万亿放量跳水定性为借机出货（05-22）；商业航天/军工被定性为情绪压制（05-27）

### 原则 4：预期你的预期——反向博弈识别做局
- 触发：有催化但持续阴跌
- 逻辑：有利好不涨 = 主力在预期散户的预期，做局砸盘洗筹（05-28）
- 配套监测：云厂商资本开支是科技行情崩盘的核心先行指标

### 原则 5：留子弹，等最后一杀
- 触发：阶段顶部确认后（如 4200 二次冲击失败）
- 逻辑："最后一杀只会迟到，不会不来"；恐慌盘集中出清 = 阶段性底部信号，不需要等最低点
- 操作：恐慌盘加仓 + 反弹 T 出（05-29）；整体降仓留子弹是震荡市最优策略（06-04）
- 反例边界：出清后量能萎缩继续阴跌（磨底形态）= 资金未认可底部，不进

### 原则 6：暴跌三问（标准应对模板）
①为什么跌（宏观触发因子+估值重定价）②跌到哪是底（技术支撑+政策底）③怎么操作（**分方向建仓，绝不"全仓抄底"或"全仓观望"**）（06-08）

### 原则 7：利率定性框架——区分"趋势降息"与"插曲降息"
- 逻辑：就业+通胀同时偏紧 → 宽松叙事是假的，2025 降息只是强加息周期的喘息，不能作为持仓依据（06-08）
- 反例边界：就业/消费数据快速恶化则加息叙事瓦解

### 原则 8：A股三重护城河 + 映射行情定位
- 底线逻辑：政策使命（上市渠道/化债/居民理财蓄水池）+ 国家队（3900 历史托底区）+ 3794 最后底线（06-08）
- 定位逻辑：A股 = AI 产业链中后端，只能做美股的"映射行情"，独立行情不存在（06-02）
- 反例边界：政策重心根本转移、国家队暂停托底，则全套支撑逻辑失效

### 原则 9：单方向仓位硬上限 + 分批纪律
- 实例：黄金长期看好（去美元化）但 15% 仓位硬上限，4300-4500 分批建仓；短期无规则洗盘中投机者大概率被收割（05-21）
- 逻辑：长期逻辑正确 ≠ 短期可以重仓；仓位上限独立于看好程度

### 原则 10：卖飞永远是对的；行业战略意义 ≠ 股价涨幅
- 逻辑：止盈后继续涨不构成错误——纪律的价值在重复执行（06-05）
- 配套：事件落地前持仓不动（中美会晤/美联储换届前观望，05-12）

## 二、实盘操作流程（信号 → 动作树）

```
[每日收盘信号输入]
    ↓
① 有无新增信息？
   无 → 维持原判，不操作（原则1）
   有 ↓
② 信息定性：宏观利率 / 板块催化 / 资金行为？
   ├─ 宏观利率类 → 趋势还是插曲？（原则7）→ 重定价高估值仓位
   ├─ 板块催化类 → 有催化但不涨？→ 做局警报（原则4）→ 不接刀
   └─ 资金行为类 → 拥挤度检查（原则2）→ 超阈值降仓
    ↓
③ 大跌日专用：暴跌三问（原则6）
   ├─ 错杀（缩量跟跌，逻辑未变）→ 持有/加仓（原则3）
   ├─ 出货（放量破位）→ 减仓走人
   └─ 恐慌出清（全线无差别+放量）→ 分方向建仓，短线品种反弹即撤（原则5）
    ↓
④ 任何建仓动作前置检查：
   仓位上限未超？（原则9）｜分批而非一次？｜短线品种撤退价位已设？（原则10）
```

## 三、持仓管理画像（板块进退逻辑，2026-06-08 快照）

| 板块 | 角色 | 进退逻辑 |
|------|------|---------|
| 电池/储能 | 中期主攻 | 跌幅充分+基本面未变 → 可建仓加仓，等下半年主升 |
| 电网设备 | 长逻辑等待位 | 算电协同+电价+厄尔尼诺三重催化，但高位未充分回调 → 等催化落地再进 |
| 商业航天/军工 | 错杀观察位 | 资金做局明显，逻辑未变 → 等企稳确认，不追跌（SpaceX IPO 是估值重塑催化） |
| 港股创新药 | 暂停位 | 调整未稳 → 当前操作无意义 |
| 恒生科技/券商 | 底仓 | 持仓等待，不加不减 |
| 创业板ETF | 纯博弈工具 | 暴跌后抄一份，反弹立即撤，绝不过夜成信仰 |
| 科技（算力/AI） | 高位警戒区 | 拥挤度与 2000 年顶点同构（06-01），仅持观察仓，不追涨 |

**通用进退节奏**：不确定性高→保持现仓观望｜信号积极→分批加仓｜获利丰厚但有风险→分次止盈。

## 四、风险管理哲学

- **最敏感**：宏观利率预期（直接重塑全球定价）> 资金做局/拥挤度 > 自身行为风险（高波动时人会乱动——他认为投资者最大的敌人是自己的焦虑）
- **规避方式层级**：①不追高 ②分方向分批 ③短线设撤退点即撤 ④极端时持币等待
- **信仰类支撑**：国家队托底（3900-3794 历史验证）——注意这是**信任度最高但可证伪性最差**的一块
- **每日复盘闭环**：大跌后必引用此前预警（"6月2日我说过"），用预判记录建立信用——这也是建预测追踪表最适配的信源

## 五、表达风格建模

- **确定性分级语言**（他的置信度就藏在措辞里）：高=「肯定的」「已经出来了」｜中=「可以考虑」「可建仓」｜低=「如果…」「也可能」
- **情绪锚定功能**：读者情绪最差时给方向锚（「不用慌」「机会都是跌出来的」），从不随波助跌
- **招牌式收尾**：「最后一杀，只会迟到，不会不来」——制造等待感，为加仓埋伏笔
- **标题情绪化但正文克制**：标题「暴跌！」「吃爽了！」是流量界面，正文操作建议始终留余地
- **自我复盘高频**：常回顾前日观点，对错都认

### 仿写校验（他不会这样写）
1. 不会给出不带仓位约束的看多/看空（永远附"几成仓""分批"）
2. 不会在信息空白日硬挤分析（宁可 200 字收工）
3. 不会用"长期价值"为短线套牢辩护（他的框架是趋势+催化剂的中周期，不是价值投资）
4. 不会绝对化预测点位（说"大概率""区间"，不说"必到"）
5. 不会忘记提示自己上次说了什么（复盘闭环是身份标识）

## 六、模型可靠性评估

### 有效场景
- 箱体震荡市（4000-4200 这类区间）：留子弹+恐慌盘加仓+反弹T出的打法收益风险比最好
- 情绪错杀频发期：原则 3 的量能判别能稳定捡到便宜筹码
- 信息过载期：原则 1 提供"不动"的合法性，避免被噪音磨损

### 失效场景
- **趋势性熊市**："等待"会系统性扩大亏损——错杀和趋势破坏在下跌前期不可分辨，他的框架默认均值回归
- **国家队规律失灵**：三重护城河是历史归纳不是保险合同，若政策选择"以时间换空间"，短线博弈变中线套牢
- **信息茧房**："无新增信息"的判断可能本身就是错的——拥挤信号常被他归类为"已知"而非"新信息"
- **映射行情失锚**：若美股 AI 叙事崩塌，A股"中后端映射"定位意味着没有独立避险能力

### 与其他体系的差异
- vs 黄哥：黄哥从结构出发拒绝预测时点；浪淘沙每天都给时点判断但用仓位纪律对冲预测误差——一个赚认知的钱，一个赚纪律的钱
- vs 纯技术派：他看位置（4000/3794）但不迷信形态，催化剂和资金行为权重更高
- vs 价值投资：持有周期以周-月计，"卖飞永远是对的"与价值投资的"永远持有"正面冲突

## 对抗步骤（反方视角）

1. **该模型的命门是"底线信仰"**：十条原则里有九条是防守纪律，唯独"3794 不破""国家队必出手"是进攻性信仰——整个系统最不可证伪的部分恰恰承担了最重的决策权重。预测追踪表 #5 是对它的第一次正式检验。
2. **日评的幸存者偏差**：他每天复盘自己的预警命中，但"说过的话"里被引用的永远是对的那部分——只有预测追踪表的全量记录（命中+落空都记）才能给出真实胜率，模型对他可信度的评估应以追踪表为准，而非他的自我复盘。
