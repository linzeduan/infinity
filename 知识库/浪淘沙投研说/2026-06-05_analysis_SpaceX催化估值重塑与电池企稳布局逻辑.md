## STEP 2 · 决策原则提炼

**原则1：卖飞永远是对的，只赚认知范围内的钱**
- 触发条件：持有期间涨幅越来越大，分批止盈压力上升
- 行动方向：以824作为第一批止盈锚点（市场讨论"牛市起点"时），此后每波涨幅越大卖得越多，最终4月全清
- 反例边界：若有更深度的基本面研究支撑（如ROE结构性改善），可适度推后止盈节点；不适用于靠逻辑驱动的长期持仓

**原则2：行业战略意义 ≠ 股价涨幅，看好逻辑不等于重仓**
- 触发条件：SpaceX IPO带来商业航天叙事升级，战略意义比新能源车可能更大
- 行动方向：关注板块估值体系重塑，但"算是淘沙客本身投资的风格"——即保持分散、不重仓主题
- 反例边界：当叙事催化+实质业绩双重确认时（如2018年特斯拉上海工厂落地），可适当提高仓位上限

**原则3：等待基本面节点确认，不抢跑**
- 触发条件：电池/锂矿方向有企稳迹象但锂价未见底
- 行动方向：观察①7月津巴布韦锂矿到港后库存数据②7月排产实际数据③储能政策落地——股价通常先于锂价见底
- 反例边界：若宏观持续利空（需求不及预期、资金面收紧），不应以"历史规律"为由提前抢跑

**原则4：高位抱团强化=预留退路，而非追入**
- 触发条件：头部个股成交额占比越来越高，抱团强化信号出现
- 行动方向：结合自身风险偏好控制仓位，不追涨；科技方向仅维持观察仓
- 反例边界：若是机构长期配置行为+业绩持续兑现，抱团本身不一定危险；需区分"短期博弈拥挤"vs"基本面驱动集中"

**原则5：顶部/底部是区间不是时间点**
- 触发条件：任何时候做高低位置判断
- 行动方向：区间内分批操作，不追求精确的最高/最低点
- 反例边界：有明确催化事件（政策发布、重大财报）时，可更精准判断阶段性节点；仅作为基础判断原则使用

**原则6：小仓位观察仓的"良心持仓"原则**
- 触发条件：看好但无法确定性给出入场信号的方向（如科技）
- 行动方向：持有万元级别观察仓，不对外"包装"成大神持仓，不用收益率制造虚假光环
- 反例边界：若有充分研究支撑且仓位管理能力足够强，可提高到正式仓位

## STEP 4 · 风险意识画像

**最敏感风险类型**：情绪拥挤风险 + 流动性撤退风险（多头强平模型）

**规避方式**：
1. 分批止盈——不贪心，分阶段锁定收益
2. 分散持仓——多个低相关方向（电池/电网/航天/黄金/创新药），不集中押单一赛道
3. 小仓位观察——高位拥挤期主动降低科技仓位，以万元级别"站队"而非重仓

**持有期望**：中期为主（有明确催化节点的事件驱动逻辑）

**认知盲区（本文未充分讨论的）**：
- 商业航天国内产业链映射的具体标的选择逻辑尚未展开
- 港股创新药"等待催化"具体等什么信号未说明
- 科技观察仓的退出条件没有明确设定

## 核心决策流程（可复用）

```
有重大海外催化（如大公司IPO/重磅事件）
  → 判断：国内板块是否存在映射逻辑？
  → 如有映射 → 评估行业战略意义（短期情绪 vs 长期基本面分离程度）
  → 仓位：情绪驱动则轻仓观察，基本面驱动则可正常布局

某赛道阶段高位后下跌
  → 识别下跌驱动：利空事件？还是资金撤退？还是基本面恶化？
  → 如为利空事件（津巴布韦锂矿），则：
    → 等待利空事件解除时间节点（7月到港）
    → 股价先于商品价格见底 → 观察止跌企稳力度 → 可建仓
  → 如为基本面恶化，则不抢跑

科技/热门板块高位抱团信号出现
  → 不追高，维持观察仓
  → 留子弹等待其他方向（电池/电网）的布局机会
```

## 反方视角

**最可能错在哪里**：作者将商业航天的"战略意义"类比为2018年特斯拉上海工厂，但新能源车的爆发核心是政策补贴+国内规模化生产降成本，商业航天在国内的商业闭环（谁付费？B端还是C端？政府订单占比？）并不清晰，战略意义≠商业可行性，估值重塑的持续性存疑。

此外，"卖飞永远是对的"是后验视角，若作者4月清仓后科技继续涨40%，这个原则在下一个类似场景中是否仍然适用取决于持仓成本和认知深度，不能直接照搬。
